# 作者序

耶稣受苦之事，我从未像写这本书时，有这样的真实、这样的体会，以及这样的领受。我的思想引领我的心，跟随耶稣沉痛的旅程走向各各他。我见证祂的受难、身体和灵性的受苦。

最令我感动的，应该是耶稣卑微身体的伟大；可以看出主虽是受苦之人，却不是失败者。在忍受残酷的一切过程中，彰显出祂的神性、祂的受苦受死，祂是伟大得胜者的表彰。

《与耶稣同往各各他》一书，主后一九三四年初版。以后曾多次再版，主后一九八九年再版销售一空后，今已绝版。我感谢出版社再印此书，内容与以前所印完全相同。

预苦期崇拜，随时间演变，现在参加的人较从前减少些。然而，基督徒如果不尊重预苦期，是一种很大的损失。惟愿本书能帮助读者，专心想得救稳固的根基。我所写的一些书，只有这本最能代表我心裡的领受和体会。

魏思洛夫（Fredrik Wisløff）

主后一九七四年，写于挪威奥斯陆
